# Hercules

Concept workspace for a gym companion app that borrows Renaissance's fast-capture interface and reshapes it around lifting sessions.

## What is here

- `ui/`: isolated React Native UI basics for the first Hercules home screen.
- `DESIGN.md`: living product and design document for collaboration.
- `DATA_MODEL.md`: v1 entity and storage spec for sessions, sets, timers, and parser traces.
- `EVALS.md`: gym-speak parsing examples and evaluation fixtures.

## Current scope

- Large central action button for voice-first set explanations.
- Rest timer panel that starts from the same capture flow.
- Session cards for set details, effort, and quick edit cleanup.

## Intent

Keep this folder decoupled from the main app until the product direction, data model, and gym-speak SLM workflow are clearer.
